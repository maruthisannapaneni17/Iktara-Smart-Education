# 🎓 IKTARA Smart Education

> AI-Powered Student Performance Prediction and Academic Analytics Platform

## 📌 Overview

IKTARA Smart Education is an intelligent educational analytics platform that uses Machine Learning to predict student academic performance based on academic, behavioral, and engagement-related factors. The system helps educators identify at-risk students early, analyze performance trends, and generate personalized recommendations for academic improvement.

The platform combines predictive modeling, data visualization, feature importance analysis, and recommendation systems to support data-driven educational decision-making.

## 🚀 Features

### 📊 Student Performance Prediction

* Predicts student performance categories:

  * Excellent
  * Good
  * Average
  * Poor
* Uses Machine Learning algorithms for accurate predictions.

### 📈 Interactive Analytics Dashboard

* Performance Distribution Analysis
* Attendance vs Marks Visualization
* Comparative Student Analytics

### 🤖 Recommendation System

Provides personalized recommendations such as:

### 🔍 Feature Importance Analysis

Identifies key factors affecting student performance:

* Attendance
* Previous Marks

### 📋 Reports & Insights

* Data-driven academic insights
* Model accuracy evaluation
* Performance analytics reports

## 💻 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/iktara-smart-education.git
cd iktara-smart-education
```

### Install Frontend Dependencies

```bash
npm install
```

### Install Backend Dependencies

```bash
pip install -r requirements.txt
```

### Start Development Server

```bash
npm run dev
```

### Run Backend

```bash
python server.py
```
